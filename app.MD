# README

## App

1. init project with yarn

    ```bash
    yarn init -y
    ```

2. add .gitignore to ignore node_modules

3. install express

    ```bash
    yarn add express
    ```

4. add yarn script to start server under license

    ```json
    "scripts": {
        "start": "node app.js"
    },
    ```

5. start server

    ```bash
    yarn run start
    ```

6. visit url - <http://localhost:4000>
